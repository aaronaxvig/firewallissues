## GPC-14612

Fixed an issue where, on Android devices with the GlobalProtect app installed, when the client certificate alias setting was set to pre-select in the managed configuration, the GlobalProtect **Choose certificate** pop-up failed to appear and the user was redirected to the captive portal (SAML) login page.
