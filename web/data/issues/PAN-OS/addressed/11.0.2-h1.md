Fixed an issue where disk space utilization was higher than expected
due to excessive logging for a KNI: Out of
memory event under a specific traffic load
condition.

issue.

```caveat
VM-Series firewalls in Microsoft Azure environments only
```

and was only recoverable by restarting the firewall.

## PAN-220921

Fixed an issue where return tunnel traffic was dropped with the
counter flow_tunnel_encap_err when
Enforce Symmetric Return was enabled in a
Policy Based Forwarding rule.


a hotplug event triggered by Azure infrastructure.

partition utilization reached 100%, which caused access to the
Panorama web interface to fail.
